## 🛠️ First-time setup

When you start tarn for the first time, you can set up a new wallet or open one you already have.

### Create a new wallet

1. Open tarn.
2. Choose Create New Wallet.
3. Enter a wallet name.
4. Set a strong password.
5. Save your recovery phrase in a safe place.
6. Finish the setup.

### Open an existing wallet

1. Open tarn.
2. Choose Open Wallet.
3. Select your wallet file.
4. Enter your password.
5. Wait for the wallet to load.

## 🔐 Keep your wallet safe

Use these simple steps to protect your Monero wallet:

- Save your recovery phrase on paper
- Keep your password private
- Do not share your wallet file
- Use a device that only you can access
- Back up your wallet after setup
- Store backups in two safe places

If you lose your recovery phrase, you may not be able to restore your wallet.

## 📤 Send XMR

To send Monero with tarn:

1. Open your wallet.
2. Click Send.
3. Paste the recipient address.
4. Enter the amount of XMR.
5. Check the details.
6. Confirm the payment.

Before you send funds, check the address one more time. A wrong address can send your XMR to the wrong place.

## 📥 Receive XMR

To receive Monero:

1. Open your wallet.
2. Click Receive.
3. Copy your wallet address.
4. Send that address to the person paying you.
5. Wait for the payment to arrive.

You can also use a payment request if tarn shows one.

## 🧩 Common problems and fixes

### The app will not open

Try these steps:

1. Right-click the app.
2. Choose Run as administrator.
3. Check if Windows blocked the file.
4. Download the latest version again.
5. Restart your computer and try again.

### The wallet does not load

Try this:

1. Make sure you picked the correct wallet file.
2. Check that your password is correct.
3. Confirm the wallet file is not damaged.
4. Open a backup copy if you have one.

### Sync looks slow

Try this:

1. Check your internet connection.
2. Leave the app open for a longer time.
3. Make sure Windows is not in battery saver mode.
4. Close other apps that use a lot of network or disk activity.
